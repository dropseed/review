# PullApprove for VS Code

A code review helper for comparing branches and tracking reviewed files.

## Features

- **Compare branches** - Select old and new branches to see changed files
- **Track reviewed files** - Check off files as you review them
- **Review notes** - Write notes and add file/line references
- **Copy as Markdown** - Export your review for sharing

## Usage

1. Click the PullApprove icon in the Activity Bar
2. Select the "old" branch (e.g., `main`) and "new" branch (e.g., `HEAD` or a feature branch)
3. Review files by clicking to open diffs
4. Check off files as you review them
5. Add notes in the Review panel
6. Use `Cmd+Shift+R` (Mac) or `Ctrl+Shift+R` (Windows/Linux) to add line references from the editor
7. Click "Copy" to copy your review as Markdown
